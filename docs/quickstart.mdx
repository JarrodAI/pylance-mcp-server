## Installation

<Steps>
  <Step title="Install via npm (Recommended)">
    ```bash
    # Install globally
    npm install -g pylance-mcp-server
    
    # Or use with npx (no installation needed)
    npx pylance-mcp-server
    ```
    
    This automatically handles all dependencies including Python packages.
  </Step>
  
  <Step title="Alternative: Install from Source">
    ```bash
    # Clone the repository
    git clone https://github.com/JarrodAI/pylance-mcp.git
    cd pylance-mcp
    
    # Install Python dependencies
    pip install -r requirements.txt
    ```
  </Step>
  
  <Step title="Test the Installation">
    ```bash
    # If installed via npm
    pylance-mcp-server
    
    # If installed from source
    python server.py
    ```
    
    The server should start without errors.
  </Step>
</Steps>

## Configure Your AI Client

<Tabs>
  <Tab title="Cursor">
    Add to `.cursor/config.json`:
    
    ```json
    {
      "mcpServers": {
        "pylance": {
          "command": "npx",
          "args": ["-y", "pylance-mcp-server"],
          "env": {
            "WORKSPACE_PATH": "${workspaceFolder}"
          }
        }
      }
    }
    ```
  </Tab>
  
  <Tab title="Claude Desktop">
    Add to `~/Library/Application Support/Claude/claude_desktop_config.json` (macOS) or `%APPDATA%\Claude\claude_desktop_config.json` (Windows):
    
    ```json
    {
      "mcpServers": {
        "pylance": {
          "command": "npx",
          "args": ["-y", "pylance-mcp-server"],
          "env": {
            "WORKSPACE_PATH": "${workspaceFolder}"
          }
        }
      }
    }
    ```
  </Tab>
  
  <Tab title="Continue.dev">
    Add to `.continue/config.json`:
    
    ```json
    {
      "mcpServers": {
        "pylance": {
          "command": "npx",
          "args": ["-y", "pylance-mcp-server"],
          "env": {
            "WORKSPACE_PATH": "${workspaceFolder}"
          }
        }
      }
    }
        }
      }
    }
    ```
  </Tab>
  
  <Tab title="Windsurf">
    Add to Windsurf MCP settings:
    
    ```json
    {
      "pylance-mcp-pro": {
        "command": "python",
        "args": ["server.py"],
        "env": {
          "WORKSPACE_ROOT": "/path/to/project"
        }
      }
    }
    ```
  </Tab>
</Tabs>

## Verify Connection

<Steps>
  <Step title="Restart Your AI Client">
    Close and reopen your AI assistant to load the new MCP configuration.
  </Step>
  
  <Step title="Test a Simple Prompt">
    Try asking your AI:
    
    > "Use Pylance MCP to show me type information for the main function in src/app.py"
    
    Your AI should use the `get_hover` tool to retrieve type signatures.
  </Step>
  
  <Step title="Verify Health Check">
    Ask your AI:
    
    > "Run a health check on the Pylance MCP server"
    
    You should see server status, version, and workspace information.
  </Step>
</Steps>
